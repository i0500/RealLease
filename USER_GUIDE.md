# RealLease 사용 설명서

## 📱 앱 설치 및 실행 (컴퓨터 몰라도 OK!)

### 1️⃣ 처음 사용하기

**[관리자가 공유한 주소]를 브라우저 주소창에 입력**
```
예시: https://reallease.vercel.app
```

### 2️⃣ 앱 설치하기 (선택사항)

브라우저에서 접속하면 **"앱 설치"** 또는 **"홈 화면에 추가"** 버튼이 나타납니다.

**컴퓨터 (Chrome, Edge)**
1. 주소창 오른쪽에 "설치" 버튼 클릭
2. 확인 클릭
3. 바탕화면에 아이콘 생성됨

**아이폰**
1. Safari에서 주소 입력
2. 하단 공유 버튼 (□↑) 클릭
3. "홈 화면에 추가" 선택
4. "추가" 클릭

**안드로이드**
1. Chrome에서 주소 입력
2. 메뉴 (⋮) 클릭
3. "홈 화면에 추가" 선택
4. "추가" 클릭

### 3️⃣ 로그인하기

1. 이메일 입력
2. 비밀번호 입력
3. "로그인" 버튼 클릭

## 💡 주요 기능 사용법

### 📊 대시보드
- 홈 화면에서 전체 현황 확인
- 계약 만료 임박 알림
- HUG 보증 만료 알림

### 📝 계약 관리
1. **새 계약 추가**
   - "계약 관리" 메뉴 클릭
   - "계약 추가" 버튼 클릭
   - 양식 작성 후 저장

2. **계약 수정**
   - 계약 목록에서 수정할 계약 클릭
   - "수정" 버튼 클릭
   - 내용 변경 후 저장

3. **계약 검색**
   - 검색창에 임차인 이름, 주소, 전화번호 입력
   - 자동으로 필터링됨

### 🔔 알림 확인
- "알림" 메뉴에서 만료 예정 계약 확인
- 우선순위별로 정리되어 표시됨
- 클릭하면 해당 계약으로 이동

### ⚙️ 설정
1. **구글 시트 연결**
   - "설정" 메뉴 클릭
   - "시트 추가" 버튼 클릭
   - 구글 스프레드시트 URL 입력
   - 저장

2. **시트 동기화**
   - 시트 옆 "동기화" 버튼 클릭
   - 구글 시트 데이터가 앱에 반영됨

## ❓ 자주 묻는 질문

**Q: 인터넷 없이도 사용 가능한가요?**
A: 한 번 접속 후에는 오프라인에서도 기본 기능 사용 가능합니다. 단, 동기화는 인터넷 필요.

**Q: 여러 기기에서 사용할 수 있나요?**
A: 네! 같은 계정으로 로그인하면 모든 기기에서 동일한 데이터 확인 가능.

**Q: 데이터는 어디에 저장되나요?**
A: 구글 스프레드시트에 저장됩니다. 안전하게 백업됨.

**Q: 앱을 업데이트해야 하나요?**
A: 자동으로 업데이트됩니다. 페이지 새로고침하면 최신 버전 적용.

**Q: 비밀번호를 잊어버렸어요**
A: 관리자에게 문의하세요.
